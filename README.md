## Unity UI Basics

### The Hierarchy
In the left pane there is the Hierarchy, this is where your `Scene`'s `GameObject`'s
will appear. You can add new ones by right-clicking in this pane 

### Game Editor/Viewer/Player
The center pane contains the player/editor/viewer. when you press play this will
switch to the `Game` tab which allows you to view and play your game in the unity editor.
to move objects around in the Scene you can use the `Scene` tab.

### The Inspector
This is the right panel. Try clicking on an object from the `Hierarchy` and see what happens.
You can edit your game objects directly from here.

### The project tab
the bottom panel contains the project tab this houses all of the files within a game.
`Assets` & `Packages` are the two main components fo a game that unity uses. Packages refers
to installations of backend code. Assets refers to things like the objects of the game, this will
contain most of your configurations, materials, objects, scrips, and more
